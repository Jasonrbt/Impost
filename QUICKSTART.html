🎭 Qui est l'Imposteur?

Guide de démarrage rapide

✓ Installation Complètement Simple

Bonne nouvelle: Vous n'avez besoin que de PHP! Aucune base de données, aucune configuration complexe.

📋 Prérequis

  • PHP 7.4 ou supérieur
  • Un navigateur web moderne
  • Une connexion internet (pour les emojis)

🚀 Démarrage (Windows)

  1. Double-cliquez sur start.bat
  2. Attendez le message "Serveur Web démarré"
  3. Ouvrez votre navigateur à http://localhost:8000
  4. Bon jeu! 🎉

🚀 Démarrage (Mac/Linux)

  1. Ouvrez un terminal dans ce dossier
  2. Exécutez:
    chmod +x start.sh && ./start.sh
  3. Ouvrez votre navigateur à http://localhost:8000
  4. Bon jeu! 🎉

🚀 Démarrage (Manuel avec Terminal)

  1. Ouvrez un terminal/invite de commande
  2. Allez au dossier du projet:
    cd C:\Users\Utilisateur\Desktop\Impost
  3. Lancez le serveur PHP:
    php -S localhost:8000
  4. Ouvrez http://localhost:8000 dans votre navigateur

💡 Changer le Port

Si le port 8000 est occupé, utilisez un autre:

php -S localhost:8001

Puis accédez à http://localhost:8001

🎮 Comment Jouer

  1. Créer/Rejoindre: Créez une partie ou rejoignez-en une avec un code
  2. Profil: Choisissez un pseudo et un avatar manga
  3. Jeu: L'imposteur essaie de deviner le mot secret
  4. Vote: Après 2 tours, votez pour éliminer le suspect
  5. Victoire: Citoyens gagnent si l'imposteur est éliminé!

⚠️ Dépannage

PHP n'est pas trouvé? Vérifiez l'installation de PHP et assurez-vous qu'il est dans le PATH.

Port 8000 déjà utilisé? Utilisez un autre port (8001, 8002, etc.)

Navigateur ne répond pas? Essayez avec un autre navigateur ou videz le cache.
