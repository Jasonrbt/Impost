🚀 DÉMARRAGE RAPIDE
────────────────────

WINDOWS:
1. Double-cliquez sur "start.bat"
2. Ouvrez http://localhost:8000

MAC/LINUX:
1. ./start.sh
2. Ouvrez http://localhost:8000

MANUEL:
1. php -S localhost:8000
2. Ouvrez http://localhost:8000

🎮 COMMENT ÇA MARCHE
──────────────────────

1. CRÉER UNE PARTIE
   → Un code unique est généré automatiquement
   → Partager le code avec les amis

2. REJOINDRE UNE PARTIE
   → Entrer le code de la partie
   → Attendre d'autres joueurs

3. PROFIL
   → Choisir un pseudo (3-15 caractères)
   → Choisir un avatar parmi 16 options

4. JEU
   → Un imposteur est sélectionné aléatoirement
   → Un mot secret est choisi
   → Chacun dit un mot en 30 secondes
   → L'imposteur essaie de deviner le mot
   → Après 2 tours → Vote pour éliminer
   → Citoyens gagnent si l'imposteur est éliminé

📝 RÈGLES DU JEU
─────────────────

1. Un joueur est sélectionné aléatoirement comme imposteur
2. Un mot secret est choisi (ex: "Banane")
3. Tous les joueurs sauf l'imposteur connaissent le mot
4. Chacun dit UN MOT qui se rapporte au mot secret
5. L'imposteur essaie de deviner en écoutant
6. Après 2 tours, tout le monde vote
7. Le plus voté est éliminé
8. CITOYENS GAGNENT si l'imposteur est découvert
9. IMPOSTEUR GAGNE si le mot est deviné ou tous les citoyens sont éliminés

🐛 DÉPANNAGE
──────────────

PHP introuvable?
→ Installez PHP ou ajoutez-le au PATH Windows

Port 8000 déjà utilisé?
→ php -S localhost:8001 (ou un autre port)

Les sessions ne fonctionnent pas?
→ Essayez dans une fenêtre incognito

CSS/images ne s'affichent pas?
→ Videz le cache du navigateur (Ctrl+Shift+Suppr)

🌐 ACCÈS DISTANT
─────────────────

Pour accéder depuis un autre ordinateur:
1. Trouvez votre adresse IP locale: ipconfig (Windows) ou ifconfig (Unix)
2. Utilisez: http://[VOTRE_IP]:8000 depuis un autre ordinateur
3. Assurez-vous que le port 8000 n'est pas bloqué par le firewall
